MS Data Science Thesis at University of Massachusetts Dartmouth (Prof. Firas Khatib). Comprehensive analysis of global trade network evolution (1988–2023) using UN Comtrade HS0 data, testing small-world properties, scale-free structure, hubs, communities, and resilience across 35+ years.

🎯 Research Objectives
Test 8 hypotheses on trade network topology and resilience:

H1-H4: Small-world properties (clustering, path length, σ coefficient)

H5-H7: Scale-free, community structure, hub centrality

H8: Shock propagation and resilience under targeted disruptions

🔬 Analysis Pipeline
text
1. DATA LOADING → Yearly trade flows (reporter → partner, HS0)
2. NETWORK CONSTRUCTION → Directed weighted graphs (countries)
3. TOPOLOGY METRICS → 
   • Nodes/Edges/Density evolution
   • Degree/strength distributions
   • Clustering coefficient (C vs Crandom)
   • Characteristic path length
   • Small-world σ = C/Cr × Lr/L
4. CENTRALITY → Degree, betweenness, eigenvector
5. COMMUNITIES → Louvain modularity (regional blocs)
6. RESILIENCE → Hub removal cascades, connectivity loss
7. VISUALIZATION → Temporal trends, distributions, networks
